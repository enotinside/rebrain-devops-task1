# NGINX

Краткий мануал по установке и запуску cервиса на системах Linux

## Операционная система

Cent OS
Ubuntu Server

### Установка

CentOS:

```
yum install nginx -y
```

Ubuntu Server:

```
apt install nginx -y
```

#### Запуск и проверка состояния

_Запуск nginx_

```
systemctl start nginx
```

_Проверка состояния_

```
systemctl status nginx
```
